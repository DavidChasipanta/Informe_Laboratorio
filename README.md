2. Exprese cada número fraccionario en notación científica:

(a) 1/500 (b) 1/2000 (c) 1/5,000,000

* 1/2000 =  5 x 10^-4
*  1/5,000,000 =2x10^-1

(a) 0.0002 (b) 0.6 (c) 7.8x10^-2

* 0.0002 = 2x10^-4
